# Компьютерная алгебра
#### Установка и запуск
1. ```git clone https://github.com/ochaplashkin/computer_algebra```
2. ```python3.7 -m pip install -r requierments.txt```
3. ```python3.7 <lection_folder>/main.py [optional arguments]```
